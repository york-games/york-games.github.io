Mac

PulseSecure VPN

The easiest method to connect is through the University's official VPN. You can set up the PulseSecure client with the instructions for Mac on the IT Services page.

SSH Tunnel

Using SSH on Mac doesn't require downloading any additional software.

  1. Register with the University SSH service. Email itsupport@york.ac.uk and ask to be added to the SSH service. You only need to do this once.
  2. Open a terminal and run the following command: ssh -nfoProxyJump="<username>@ssh.york.ac.uk" <server> -L<port>:localhost:<port>, replacing:
    • <username> with your York ITS username (eg abc500). This is the same as the first part of your york.ac.uk email address
    • <server> with the server IP that you are connecting to (eg mc.york.games)
    • <port> with the port (eg 25565)
    After you input your password correctly, you can close the terminal and the tunnel will remain open until you disconnect from the network or shut down.
  3. When specifying the server address in-game, use localhost instead of the york.games server address.

The SSH service accepts public key authentication, so you can add an SSH public key with ssh-copy-id <username>@ssh.york.ac.uk to connect without specifying your password each time.
