MacOS

1. Set up SSH tunnel

If you have not registered for the University SSH service you will need to do so. Open Terminal and run the following command, replacing <itsusername> with your University username. (What does this do?) 
ssh -NfL 25565:apollohacksoc.york.ac.uk:25565 <itsusername>@ssh.york.ac.uk
You can close Terminal once you have logged in. You will need to re-run this command every time you log in.

2. Connect to localhost

Add a new server in Minecraft with the address localhost. You should be able to connect to it
